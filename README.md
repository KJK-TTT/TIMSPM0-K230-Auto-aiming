## v2.0 更新内容(master branch)

在第二版系统中，对硬件结构与软件系统进行了升级和优化，主要包括以下内容：

- 重新设计系统硬件架构，完成 **电源板与控制板 PCB 设计**，制定整体电源方案，并设计编码器电机、步进电机及传感器驱动电路。
- 将原有 **舵机二维云台升级为步进电机二维云台**，通过步进电机驱动提高云台控制精度与稳定性。
- 新增 **串口屏 UI 人机交互界面**，用于显示系统运行状态、参数信息，并支持任务模式切换与参数设置。
- 基于主控 **TI MSPM0G3507** 搭建嵌入式控制平台，实现底盘运动控制与云台控制。
- 设计 **云台闭环控制算法**，根据视觉模块输出的目标偏差计算误差，采用 PID 控制驱动步进电机，实现激光稳定瞄准目标。
- 设计 **底盘差速控制算法**，结合编码器速度闭环、灰度循迹算法与陀螺仪偏航角前馈补偿，实现小车稳定循迹运动。
- 基于 **UART 通信协议** 封装数据包，实现视觉模块、陀螺仪、串口屏与蓝牙模块之间的数据通信。
